# WS2811 と WS2812B の違い

## 概要

WS2811 は外付け RGB LED を駆動するための独立したドライバ IC です。
一方で WS2812B は RGB LED とドライバ回路を 1 つの 5050 パッケージに統合した素子です。

## 主な違い

| 項目 | WS2811 | WS2812B |
|------|--------|----------|
| 構成 | 外付け LED を駆動するドライバ IC | ドライバ IC と RGB LED の一体型 |
| 動作電圧 | 5[V] または 12[V]（構成による） | 5[V] 固定 |
| 駆動方式 | 定電流出力（約18.5[mA]） | 内蔵定電流ドライバ |
| 制御方式 | 1 線式（NRZ）、400/800[kbps] | 1 線式（NRZ）、800[kbps] 固定 |
| ピクセル単位 | 1 IC で RGB3 直列（12V 構成時） | 1 パッケージ 1 ピクセル |
| 実装形態 | IC + 外部 LED | 5050SMD LED モジュール |
| 放熱・保守性 | 高（LED 交換可） | 一体構造のため交換不可 |

## 利点と用途

- **WS2811**  
  長距離伝送や 12[V] 系の LED ストリップに適しています。  
  LED と IC を別に配置できるため、電圧降下に強く、放熱設計が容易です。

- **WS2812B**  
  小型・軽量で、細かいピクセル制御を行いたい用途に適します。  
  5[V] 系で扱いやすく、Arduino や ESP32 などから直接制御できます。

## 信号タイミングの違い

どちらも 1 線式でデータを直列転送しますが、ビットごとのパルス幅が異なります。  
WS2811 は 400[kbps]/800[kbps] モードがあり、WS2812B は 800[kbps] 固定です。  
混在すると波形タイミングの差で誤動作します。

## まとめ

WS2811 は「外部 LED 用のドライバ IC」、WS2812B は「LED 内蔵のアドレス指定可能 LED」です。  
電源電圧・実装構成・発熱対策を考慮して選定することが重要です。
